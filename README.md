Projeto pessoal para estudo e prática de desenvolvimento backend utilizando Python e Flask.

## Descrição
Esse projeto consiste no desenvolvimento de uma API REST para estudo de conceitos de backend, incluindo organização de endpoints, manipulação de dados, validação de requisições e respostas, e estruturação de regras de negócio.

O objetivo é praticar boas práticas de desenvolvimento backend e preparar base para integração com banco de dados.

## Tecnologias utilizadas
- Python
- Flask
- APIs REST
- JSON
- Requests HTTP
- Git / GitHub
- SQL / MySQL (em progresso)

## Funcionalidades (em desenvolvimento)
- CRUD de entidades (ex: products, sellers, sales)
- Estruturação de endpoints REST
- Organização em camadas (controllers, use cases, repositories)
- Testes básicos de requisições

## O que estou explorando com este projeto
- Estruturação de APIs REST
- Consumo e validação de dados via HTTP
- Organização de código backend
- Versionamento com Git
- DDD

## Status
Em desenvolvimento (integração com o banco feita e em teste)
